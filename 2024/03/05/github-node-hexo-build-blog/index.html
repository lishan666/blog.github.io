    github+node+hexo搭建静态博客

    网站设计 hexo
    发布时间 : 2024-03-05 00:21
    字数:442 阅读 :
    评论:
    1. 1 申请github仓库
    2. 2 安装git并设置ssh秘钥
    3. 3 安装node.js
    4. 4 安装hexo
    5. 5 发布文章

    1 申请github仓库

    1、登录github

    GitHub: Let’s build from here · GitHub

    2、新建仓库

    在github中选择Create a new repository

    仓库名为：用户名.github.io，例如：lishan666.github.io

    仓库类型选择Public，并勾选Add a README file选项框

    3、博客地址

    启用的博客地址为：https://用户名.github.io

    例如：https://lishan666.github.io

    2 安装git并设置ssh秘钥

    1、安装git

    Git - Downloads (git-scm.com)

    2、设置git参数

    git config --global user.name "name"
git config --global user.email "email"

    3、生成本地秘钥

    ssh-keygen -t rsa -C "email"

    点击3次回车后成功创建本地公钥id_rsa.pub、私钥id_rsa

    打开id_rsa.pub文件，复制本地公钥内容

    4、github添加本地公钥

    登录github，进入 Settings 页面

    选择 SSH and GPG keys，点击 New SSH key

    粘贴复制的id_rsa.pub内容，点击Add SSH key

    5、本地添加github公钥

    在本地连接github服务器

    ssh -T git@github.com

    根据提示输入yes，将服务器的公钥保存到本地

    3 安装node.js

    1、安装node

    Node.js (nodejs.org)

    2、测试是否安装成功

    node -v
npm -v

    4 安装hexo

    1、安装hexo

    npm install -g hexo-cli

    2、初始化

    hexo init

    3、安装组件

    npm install

    4、生成页面

    hexo g

    5、启动预览

    hexo s

    浏览器访问：http://localhost:4000

    5 发布文章

    1、安装hexo-deployer-git

    npm install hexo-deployer-git --save

    2、修改 _config.yml

    deploy:
  type: git
  repository: git@github.com:用户名/用户名.github.io.git
  branch: main

    3、创建文章

    hexo new "My Post"

    在 source/_posts文件夹中会出现一个 My Post.md 文件，编辑该文件

    4、生成页面并预览

    hexo g
hexo s

    5、发布本地文章到github

    hexo d

    浏览器访问：https://用户名.github.io访问博客
